## Overview

Sober Mindset is a simple, focused web experience designed to support individuals on their journey toward sobriety.

The goal of this project is to provide clarity, motivation, and practical tools for people who want to build a better, more present life.

This is not just a website — it’s a starting point.

---

## Purpose

Many people struggle silently with alcohol or substance use.
This project aims to:

* Provide encouragement and perspective
* Offer simple, actionable steps toward sobriety
* Create a calm, structured space for reflection and growth
* Make support more accessible

---

## Current Features

* Clean landing page with a strong message
* Hero section with motivational quote
* “First 3 Steps to Sobriety” framework
* Structured breakdown of each step
* Call-to-action for finding meetings

---

## Planned Features

### Daily Reflections (AA)

A section displaying daily readings and reflections inspired by AA materials.

---

### Daily Activities

Rotating suggestions for newly sober individuals, such as:

* Going for a walk
* Journaling
* Attending a meeting
* Trying a new hobby

---

### Daily Affirmation & Gratitude

A section dedicated to:

* Daily affirmations
* Gratitude-based reflections or prayers

---

### Online Meeting Access

Quick-access links to:

* Current online AA meetings
* Upcoming online meetings

Designed for immediate support when needed most.

---

### Local Meetings (Location-Based)

A feature that:

* Detects user location (with permission)
* Shows nearest meetings
* Displays schedules and details
